# Interpreter

This project is a Java-based interpreter that reads a custom bytecode language and executes it.

## Features
- Supports float, bool, string, and vector3 types
- Arithmetic operations (+, -, *, /, %)
- Comparisons (>, <, ==, etc.)
- Control flow (jump, jumpif, labels)
- Function calls (print)

## How to Run
```bash
javac Interpreter.java
java Interpreter sample.txt
```
